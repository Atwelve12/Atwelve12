[^jpop]:该音乐类型融合摇滚、R&B、舞曲等元素，通过电视主题曲、卡拉OK等媒介传播。1990年代小室哲哉创作体系与视觉系乐团崛起，山下达郎开创City Pop类型，反映经济繁荣期都市生活体验.
[^夜鹿]:是一支由才华横溢的作曲家n-buna与独特嗓音的歌手suis携手组成的日本摇滚乐团。该乐队以“夜”为主题，融合摇滚、J-Pop，擅长通过专辑构建完整故事，乐评称赞n-buna的歌词充满隐喻，suis的嗓音可塑性强，使音乐兼具文学性与情感深度。他们的名字源自首张迷你专辑《夏草が邪魔をする》中的一首歌《雲と幽霊》，歌词「夜しかもう眠れずに」中提取的灵感。自2017年成立以来，ヨルシカ便以其独特的音乐风格迅速崛起。 2017年4月，他们通过网络发布了首支歌曲《靴の花火》的音乐录影带，随后在6月发行了首张迷你专辑《夏草が邪魔をする》。2019年4月，他们的首张录音室专辑《所以我放弃了音乐》问世，紧接着在8月发行了第二张录音室专辑《エルマ》。2020年2月，ヨルシカ荣获第34届日本金唱片大奖“五大最佳新人（日本音乐）”奖项，这是对他们音乐才华的肯定。 2021年1月，他们举行了“ヨルシカ LIVE‘前世’”线上演唱会，并在27日发行了第三张迷你专辑《创作》。2022年3月，他们踏上了“ヨルシカ LIVE TOUR 2022‘月光 再演’”巡回演唱会的征程。2023年1月，他们再次举行“ヨルシカ LIVE 2023‘前世’”演唱会，并在4月5日发行了第四张专辑《幻灯》。 值得一提的是，他们的单曲《月光浴》被选为日本动画电影《大雪海的卡纳：星球的信者》的主题曲，展现了他们在音乐领域的广泛影响力。2024年8月28日，他们参与了专辑《もしも生まれ変わったならそっとこんな声になって》的制作，继续为乐迷带来更多惊喜。
